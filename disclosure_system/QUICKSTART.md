## 기능별 사용 방법

### 📖 공시규칙 검색

#### 키워드로 검색
1. 왼쪽 메뉴에서 "📖 공시규칙 검색" 선택
2. "키워드" 탭에서 검색어 입력 (예: "계약", "투자", "증자")
3. 관련 규칙 자동 검색 및 표시

**예시**:
- "계약" 검색 → "중요 계약 체결" 규칙 표시
- "투자" 검색 → "유가증권 취득", "타법인 출자" 규칙 표시

#### 카테고리로 검색
1. "카테고리" 탭 선택
2. 카테고리 선택 (계약, 투자, 증자, 부채, 인사, 자산 등)
3. 해당 카테고리의 모든 규칙 표시

#### 기준 지표로 검색
1. "기준 지표" 탭 선택
2. 기준 지표 선택:
   - `sales_ratio`: 매출액 기준
   - `equity_ratio`: 자기자본 기준
   - `asset_ratio`: 자산 기준
3. 해당 지표를 사용하는 모든 규칙 표시

### 🧮 공시 판정 계산

#### 개별 계산 (Single Calculation)

**단계 1: 재무 지표 입력**
```
- 매출액: 10,000,000,000 (100억)
- 자산총액: 50,000,000,000 (500억)
- 자기자본: 20,000,000,000 (200억)
- 유동자산: 15,000,000,000 (150억)
- 유동부채: 10,000,000,000 (100억)
```

**단계 2: 거래 정보 입력**
```
- 거래액: 1,500,000,000 (15억)
- 시장: KOSPI
```

**단계 3: "🔍 공시 판정 계산" 클릭**

**단계 4: 결과 확인**
```
예상 결과:
- 중요 계약 체결: 🚨 공시 대상 (거래액이 기준액의 150%)
- 유가증권 취득: 🚨 공시 대상 (거래액이 기준액의 100%)
- 타법인 출자: ✅ 공시 미대상 (거래액이 기준액의 40%)
```

#### 일괄 계산 (Batch Calculation)

**단계 1: 재무 지표 입력** (동일)

**단계 2: 거래 개수 설정**
```
- 거래 개수: 3개
```

**단계 3: 거래별 정보 입력**
```
거래 1: 거래액 1,000,000,000 (10억), 카테고리 "계약"
거래 2: 거래액 2,000,000,000 (20억), 카테고리 "투자"
거래 3: 거래액 5,000,000,000 (50억), 카테고리 "M&A"
```

**단계 4: "📊 일괄 계산 실행" 클릭**

## 판정 결과 해석

### 🚨 공시 대상 (Disclosure Required)
```
거래액 ≥ 기준액
```
- **의미**: 즉시 공시 의무 발생
- **예시**: 거래액 15억이 기준액 10억 이상 → "공시 대상"
- **조치**: 즉시 공시 부서에 보고

### ⚠️ 검토 필요 (Review Required)
```
기준액 × 80% ≤ 거래액 < 기준액
```
- **의미**: 기준에 근접하여 재검토 필요
- **예시**: 거래액 8억이 기준액 10억의 80% → "검토 필요"
- **조치**: 재무/공시 담당자와 협의 필요

### ✅ 공시 미대상 (No Disclosure)
```
거래액 < 기준액 × 80%
```
- **의미**: 공시 의무 없음
- **예시**: 거래액 3억이 기준액 10억 미만 → "공시 미대상"
- **조치**: 특별 조치 불필요

## 기준 금액 계산 예제

### 예제 1: 중요 계약 체결
```
규칙: 매출액의 10% 이상
재무 지표: 매출액 = 100억
기준액 계산:
  기준액 = 100억 × (10 / 100) = 10억

테스트 거래:
  - 거래액 15억 → 150% → 🚨 공시 대상
  - 거래액 8억 → 80% → ⚠️ 검토 필요
  - 거래액 5억 → 50% → ✅ 공시 미대상
```

### 예제 2: 타법인 출자
```
규칙: 자산의 5% 이상
재무 지표: 자산총액 = 500억
기준액 계산:
  기준액 = 500억 × (5 / 100) = 25억

테스트 거래:
  - 거래액 40억 → 160% → 🚨 공시 대상
  - 거래액 20억 → 80% → ⚠️ 검토 필요
  - 거래액 10억 → 40% → ✅ 공시 미대상
```

### 예제 3: 회사 인수
```
규칙: 자기자본의 50% 이상
재무 지표: 자기자본 = 200억
기준액 계산:
  기준액 = 200억 × (50 / 100) = 100억

테스트 거래:
  - 거래액 150억 → 150% → 🚨 공시 대상
  - 거래액 80억 → 80% → ⚠️ 검토 필요
  - 거래액 50억 → 50% → ✅ 공시 미대상
```

## 자주 묻는 질문 (FAQ)

### Q1: 거래액 입력 시 주의사항은?
**A**: 거래액은 **원(₩)** 단위로 입력하세요.
- 100만 원 = 1000000
- 1억 원 = 100000000
- 10억 원 = 1000000000

### Q2: 판정이 "검토 필요"일 때 어떻게 해야 하나?
**A**: 재무/공시 담당 부서와 협의하여 다음을 검토하세요:
- 거래의 중요성
- 기타 정성적 판단
- 선례 및 규정

### Q3: 여러 규칙에 걸치는 거래는?
**A**: 시스템이 자동으로 모든 적용 규칙을 검토합니다.
- 예: "계약"이면서 "투자"인 거래는 양쪽 규칙 모두 적용
- 결과에서 "공시 대상" 항목이 하나라도 있으면 공시 필요

### Q4: 재무 지표는 어디서 구하나?
**A**: 최신 감사보고서(정기보고)에서 확인:
- 재무제표 > 대차대조표 (자산, 부채, 자본)
- 손익계산서 (매출액)
- 주석 (누적 결손금)

### Q5: 시장(KOSPI/KOSDAQ)을 잘못 선택했으면?
**A**: 다시 계산하시면 됩니다.
- 시장별로 다른 규칙이 적용될 수 있습니다.
- 상장사의 해당 시장을 선택하세요.
